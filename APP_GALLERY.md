# 🖼️ NEPSE Analysis App Gallery

Visual showcase of the NEPSE Analysis platform across multiple stacks.

## 🖥️ Desktop Analytics (Python)
The powerhouse of the platform, offering deep technical indicators and advanced portfolio metrics.

![NEPSE Dashboard Mockup](C:\Users\xxixw\.gemini\antigravity\brain\82279ab3-a95c-4935-851e-26d7ebb25de3\nepse_dashboard_mockup_1775875011865.png)
*Figure 1: Conceptual roadmap for the premium glassmorphic dashboard interface.*

### Key Features:
- **Interactive Matplotlib/Seaborn Charts**: Zoom, pan, and analyze with sub-plots for momentum oscillators.
- **Enterprise Settings**: Comprehensive control over cache, data age, and backup rotation.
- **Multi-Source Fetching**: Reliable data from NEPSE and Yahoo Finance fallback.

---

## 📱 Mobile App (Flutter)
A modern, wireless-ready mobile experience designed for the Node + Prisma backend.

### Interfaces:
- **Dashboard**: Real-time summary of portfolio and market health.
- **AI Brief**: Automated market analysis powered by the AI Analytics engine.
- **Settings**: Seamless API base URL configuration for local or remote backends.

## 🌐 Web Dashboard (Dart)
A future-ready web interface serving as a lightweight alternative to the desktop tool.
